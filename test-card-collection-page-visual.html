üìã Detailed Checklist

Header Section

  • Heart icon displays
  • "12 Cartas Especiais" title shows
  • Sender name displays correctly
  • Recipient name displays correctly
  • Instructions text is visible
  • Header is sticky on scroll

Card Grid

  • All 12 cards display
  • Cards are in correct order (1-12)
  • Lock/Unlock icons show correctly
  • Card numbers display
  • Card titles display
  • Status badges show (Aberta/Fechada)
  • Visual distinction between opened/unopened

Card Opening

  • Click on unopened card shows confirmation
  • Confirmation modal displays correctly
  • Warning message about one-time opening
  • Cancel button works
  • Open button opens the card
  • Modal shows full content
  • Photo displays (if available)
  • Message text displays
  • Music plays (if available)
  • Falling emojis effect on first open
  • Close button works
  • ESC key closes modal

After Opening

  • Card shows as "Aberta"
  • Card has gray/muted appearance
  • Unlock icon displays
  • Cannot reopen with full content
  • Shows opening date/time

üé¨ Test Scenarios

Scenario 1: First-Time Recipient
  1. Visit page with fresh slug
  2. See all 12 unopened cards
  3. Read sender information
  4. Click first card
  5. Confirm opening
  6. Watch animation and falling emojis
  7. View full content
  8. Close modal
  9. See card marked as opened
Scenario 2: Returning Recipient
  1. Visit page with existing slug
  2. See mix of opened/unopened cards
  3. Try to open already-opened card
  4. See limited content (no photo, no message)
  5. Open another unopened card
  6. See full content with animation
Scenario 3: Error Handling
  1. Visit page with invalid slug
  2. See friendly error message
  3. Verify no console errors
